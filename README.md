Procat MAT Anket Asistanı Gizlilik Politikası

Bu uzantı (Procat MAT Anket Asistanı), kullanıcıların kişisel verilerini toplamaz, kaydetmez, sunuculara göndermez veya üçüncü taraflarla paylaşmaz.

Veri İşleme:
Uzantı, sadece kullanıcının açık isteğiyle (butona tıklama veya klavye kısayolu), ilgili yönetim panelindeki rezervasyon bilgilerini (isim, telefon, tutar vb.) okur ve kullanıcının tarayıcısının "Local Storage" (Yerel Depolama) alanına geçici olarak kaydeder. Bu veriler, sadece kullanıcının hedef anket formuna yapıştırması amacıyla kullanılır ve işlem sonrasında veya tarayıcı önbelleği temizlendiğinde silinir.

İletişim:
Uzantı, herhangi bir harici sunucu veya API ile iletişim kurmaz. Tüm işlemler kullanıcının cihazında yerel olarak gerçekleşir.
